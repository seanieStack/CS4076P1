# Aberrations  
## Actions clients can request
`ac` -> add class  
`rc` -> remove class  
`ds` -> display schedule  
`st` -> stop (close connection)  

## Server responces
`ttf` -> timetable full  
`cs`-> class added  

## Example messages from client -> server
`ac CS4076 09:00-10:00 monday CS4005B`  
`st`
